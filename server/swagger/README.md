# Swagger Documentation

This folder contains the OpenAPI 3.0 specification for the Game Show API.

## Files

- `openapi.yaml`: The OpenAPI specification file describing all API endpoints.

## Usage

You can use this YAML file to:

- Generate API documentation with tools like Swagger UI or Redoc.
- Generate client SDKs using OpenAPI generators.
- Validate API requests and responses.

## Viewing the Docs

If the server is running with Swagger UI enabled, visit `http://localhost:4000/docs` to view the interactive documentation.

Alternatively, you can upload the `openapi.yaml` file to [Swagger Editor](https://editor.swagger.io/) or [SwaggerHub](https://swaggerhub.com/) to view and edit the spec.
